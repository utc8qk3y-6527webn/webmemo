# webmemo
A simple, lightweight, and practical editor enabling plain text creation and editing, along with local loading and saving, directly on the web.

Described on one page.
Operates solely with the (webmemo.html) file.

Supported formats:
.txt, .htm, .html, .css, .js, .xml.
Other formats will result in errors.

The LOAD button resets to initial values.
The NONE button clears everything.

With a little refinement, you can create a text editing tool tailored to diverse needs.
